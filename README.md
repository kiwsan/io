# IO

## Quick start

Build and run using dockerfile:

```bash
$ docker build -f Dockerfile -t kiwsan-goapp . 
```

```bash
$ docker run --rm -it -p 8000:8000 kiwsan-goapp
```

Build and run using docker-compose:

```bash
$ docker-compose up -d
```

```bash
$ curl http://localhost:8000
```
